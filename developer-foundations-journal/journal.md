What is a computer? A computer is an electronic device for storing and processing data, typically in binary form, according to instructions given to them

What is software? A software is a collection of instructions, data, or programs used to operate computers and execute specific tasks.

What is a program? A program is a specific executable instance of software.

Why programming is hard? Programming is difficult because you are designing exact systems for an unforgiving machine, while managing complexity, ambiguity, and future change all at the same time.
